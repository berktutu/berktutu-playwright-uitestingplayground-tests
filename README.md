# Playwright UI Testing Playground Tests

The purpose of this exercise is to practice Playwright and improve UI test automation skills, including handling flaky scenarios.

## Live Version

The website is available at https://uitestingplayground.com.

## Prerequisites

- Node.js
- npm

## Usage

1. Clone the repository and then run

```bash
npm install
```

2. Install Playwright

```bash
npx playwright install
```

3. Run tests

```bash
npx playwright test
```

## What I Tried to Do

- Use user-facing locators as much as possible
- Use locators like XPath only when explicitly required
- Follow the exact instructions provided by UITestingPlayground
- Keep test logic simple and focused
- Strengthen my Playwright automation skills

## What I Learned

- Handling flaky UI behavior with proper waits and assertions
- Using different types of locators
- Playwright methods that I had not used before
- Handling elements that change in the DOM
- Running and debugging tests
- Understanding the differences between Chromium, WebKit, and Firefox
